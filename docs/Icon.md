# `<Icon />`

Use the `<Icon />` component for SVG icons.

```jsx
<Icon name="check" size={48} color="green" />
```

| Prop  | Type   | Description                     |
| ----- | ------ | ------------------------------- |
| name  | string | icon key name                   |
| size  | number | width and height in pixels      |
| color | string | a color key from `theme.colors` |
